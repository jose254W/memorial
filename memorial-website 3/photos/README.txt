PHOTOS FOLDER
=============

Place all your photos here (JPG, PNG, WEBP).

Then update data.json like this:

  "photos": [
    { "url": "photos/james-beach.jpg", "caption": "At the beach, 2022" },
    { "url": "photos/graduation.jpg",  "caption": "Graduation day" }
  ]

For the portrait photo in the hero, update:

  "profilePhoto": "photos/portrait.jpg"
